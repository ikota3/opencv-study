# OpenCV

## RGB

赤，緑，青

|  Red  | Green | Blue  |
| :---: | :---: | :---: |
| 0-255 | 0-255 | 0-255 |

## HSV

色相，彩度，明度

|  Hue  | Saturation | Value  |
| :---: | :--------: | :----: |
| 0-359 |   0-100%   | 0-100% |

OpenCV では，HSV の範囲は上記とは異なり，以下の通りとなる

|  Hue  | Saturation | Value |
| :---: | :--------: | :---: |
| 0-179 |   0-255    | 0-255 |

## 座標軸の原点は左上

画像の原点は右下ではなく，左上となる．

## 画素の順番は\(B, G, R\)の順番

RGB ではなく，BGR となる．

## img.shape からは(Y, X, COLOR)

(X, Y) の順ではなく，(Y，X) となる
